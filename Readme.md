# BOOK INFO STORE

<p> You can add books information and save it in database. </p>
<p> Filter/Search them by: Title, Author, Year, or Publisher. </p>
<p> Also view, update and delete records. </p>

## Tools
<p> 1. View All </p>
<p> 2. Search </p>
<p> 3. Add Entry </p>
<p> 4. Update Entry</p>
<p> 5. Delete Entry</p>
<p> 6. Close </p>
